# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-02-19

### Added

- Initial release of NIS2 Conditional Access Policies repository
- 22 Conditional Access policies (CA000 through CA022)
- 4 App Protection (MAM) policies for iOS, Android, and Windows
- 3 Global Secure Access configuration templates
- 5 Named Location templates for USA, EMEA, APAC, trusted IPs, and blocked countries
- Complete NIS2 Directive Article 21 mapping documentation
- Comprehensive getting started guide with step-by-step import instructions
- Regional considerations documentation for USA, EMEA, and APAC
- Required Entra ID groups documentation with dynamic membership rules
- MAM-specific guide for iOS/iPadOS device management without MDM
- Global Secure Access setup guide
- Troubleshooting guide for common issues
- FAQ document
- Customization guide
- Optional PowerShell helper scripts for validation and backup
- CONTRIBUTING.md with contribution guidelines
